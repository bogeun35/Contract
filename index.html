정산 관리가 더 쉬워졌습니다
매출 확인부터 세금계산서 발행, 입금까지
한 곳에서 처리하세요
매출 정산 확인 및 세금계산서 발행
매출 정산 진입 → 정산내역 확인 → 상세 확인 → 세금계산서 발급 → 입금 처리
  1. 좌측 메뉴에서 [매출 정산]을 선택합니다.
    [식권대장 사장님] 사이트에 로그인 후, 좌측 메뉴의 [매출 정산]을 클릭합니다.
    식권대장 사장님 > 좌측 메뉴 > [매출 정산] 클릭
  2. 내 모든 사업장(제휴점)의 정산내역을 확인합니다.
    진입 시 조회기간에 해당하는 전체 정산내역이 표시됩니다.
    정산월, 시작일/종료일, 사업자명, 매출금액, 수수료, 정산(예정)금액, 정산 확정, 증빙 처리, 입금 처리 상태를 한눈에 확인할 수 있습니다.
    매출 정산 > 정산내역 목록 (정산 확정 / 증빙 처리 / 입금 처리 상태 확인)
  3. 정산서 상세에서 거래내역, 사업자 정보를 확인합니다.
    정산 항목을 클릭하면 매출 상세 페이지로 진입합니다.
    정산서 개요(진행 상태, 매출 기간, 정산 금액, 입금 일정), 사업자 정보를 확인하고,
    하단 탭에서 거래 내역 / 증빙 내역 / 입금/이체 내역을 각각 조회할 수 있습니다.
    매출 상세 > 정산서 개요 / 사업자 정보 / 증빙 내역 확인
  4. 확인이 완료되면 우측 상단 [세금계산서 발급] 버튼을 클릭합니다.
    매출 상세 페이지 우측 상단의 [세금계산서 발급] 버튼을 클릭하면 발급 확인 팝업이 나타납니다.
    "매출 증빙내역의 세금계산서 정보를 확인하셨습니까?" 메시지를 확인한 뒤 [확인]을 선택하면 세금계산서가 발급됩니다.
    매출 상세 > 우측 상단 [세금계산서 발급] 버튼 클릭
  5. 증빙 마감일 내 완료 시 입금예정일에 실제 입금이 진행됩니다.
    세금계산서 발급과 정산 확정이 완료되면, 입금예정일에 맞춰 실제 입금 처리가 진행됩니다.
    입금 처리 상태는 매출 정산 목록의 [입금 처리] 항목 또는 매출 상세의 [입금/이체 내역] 탭에서 확인할 수 있습니다.
    매출 상세 > 입금/이체 내역 확인
매출 금액은 매일 1회 업데이트 됩니다. 정산에 이상이 있는 경우 하단의 1:1 문의를 이용해주세요.
고객사와 직접거래가 있는 경우 정산금액에서 제외되어 노출됩니다.
그룹 정산(프랜차이즈)으로 진행되는 경우 정산서가 노출되지 않습니다.
공동인증서 등록 방법
[내 계약 정보] 진입 → [공동 인증서 등록] 클릭 → 바로빌 인증서 등록 → 등록 완료
  1. 좌측 메뉴에서 [내 계약 정보]를 선택한 뒤, 인증서 정보 영역의 [공동 인증서 등록] 버튼을 클릭합니다.
    내 계약 정보 페이지에서 인증서 정보 영역을 확인합니다.
    증빙 발행 유형이 [전자세금용 인증서 등록]인 경우, 우측의 [공동 인증서 등록] 버튼을 클릭하세요.
    내 계약 정보 > 인증서 정보 > [공동 인증서 등록] 버튼 클릭
  2. 바로빌 공동인증서 등록 팝업에서 [인증서 등록]을 진행합니다.
    [공동 인증서 등록] 클릭 시 바로빌 인증서 등록 팝업이 열립니다.
    팝업 내 [사용가능 인증서 확인] 버튼을 클릭하여 PC에 저장된 인증서 목록을 확인하고,
    등록할 인증서를 선택한 뒤 [인증서 등록] 버튼을 클릭하면 등록이 완료됩니다.
    인증서 등록 전, 반드시 사용 가능한 인증서인지 확인해 주세요.
    등록 가능: 전자세금용 / 사업자 범용 / 바로빌 특목용
    등록 불가: 개인범용(4,400원), 뱅킹용, 은행용, 증권용, 보험용 등
    바로빌 공동인증서 등록 팝업 > [사용가능 인증서 확인] > [인증서 등록]
  3. 새창에서 공동인증서 등록이 완료되면, [닫기] 버튼을 클릭합니다.
    등록 완료 후 "새창에서 공동인증서 등록이 완료된 다음 [닫기] 버튼을 클릭해주세요" 안내에 따라 [닫기]를 선택합니다.
    내 계약 정보의 인증서 등록 상태가 [등록 완료]로 변경되었는지 확인해 주세요.
이미 바로빌에 공동인증서를 등록하신 경우, 연동되어 그대로 사용 가능합니다.
공동인증서 만료일 전까지 세금계산서 발행이 가능하며, 만료 시 재등록이 필요합니다.
